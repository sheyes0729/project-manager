## 🐦 简介

> `Project Manager`，一个基于 Electron + Vue3 + Typescript 开发的PC端项目管理器，主要包括项目扫描、IDE绑定、Node版本管理、TodoList等功能，目前仅支持Windows

## 🎉 启动

```sh
  # 克隆项目
  $ git clone https://github.com/sheyes0729/project-manager.git

  # 安装依赖
  $ pnpm i

  # 启动项目
  $ pnpm dev

  # 生成项目图标
  $ pnpm create:icons

  # 代码样式
  $ pnpm lint

  # 项目打包
  $ pnpm build:win32
```

## 🤔️ 常见问题

### **Q：使用.exe文件安装时安装失败**
> 安装时选择安装方式为`为所有用户安装`
